# Handwritten Text Recognition for Bank Withdrawal Slips 
A final year project to recognize handwritten text from bank withdrawal slips using CRNN and store results in MySQL. 
 
## Project Structure 
- **data/**: Raw and processed images, labels. 
- **src/**: Source code for preprocessing, model, database, and utilities. 
- **notebooks/**: Jupyter notebooks for analysis. 
- **tests/**: Unit tests. 
- **docs/**: Documentation. 
- **logs/**: Training logs. 
 
## Setup 
1. Install dependencies: Python, TensorFlow/PyTorch, MySQL. 
2. Run preprocessing scripts. 
3. Train the CRNN model. 
4. Store results in MySQL database. 

## To Run 
1. python src/app.py
